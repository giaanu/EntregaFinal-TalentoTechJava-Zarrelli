# 🛍️ Ocarina Store – React App

**Ocarina Store** es una aplicación frontend desarrollada con **React**, que representa una **versión inicial** de un proyecto de e-commerce.  
Este repositorio funcionó como **base y punto de partida** para el desarrollo de proyectos posteriores más completos, incorporando mejoras funcionales y de arquitectura.

---

## 🚀 Características

- Aplicación desarrollada con React
- Estructura inicial de tienda online
- Componentes reutilizables
- Organización modular del código
- Interfaz construida con JavaScript y CSS
- Flujo base de navegación entre vistas

---

## 🛠️ Tecnologías utilizadas

- React.js  
- Vite  
- JavaScript  
- HTML5  
- CSS3  

---

## 🎯 Objetivo del proyecto

Este proyecto fue desarrollado con un enfoque **educativo**, con el objetivo de:

- Comprender la estructura de una aplicación React
- Practicar la creación y reutilización de componentes
- Establecer una base para proyectos más avanzados
- Aplicar conceptos fundamentales de frontend moderno

**Ocarina Store** sirvió como base para la construcción de un proyecto posterior más completo, incorporando nuevas funcionalidades y mejoras en la lógica y arquitectura.
